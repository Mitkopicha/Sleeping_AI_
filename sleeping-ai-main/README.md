# Sleeping AI is a web app that generates and narrates AI-generated bedtime stories for adults, helping users improve their sleep quality.

# Features
- AI-generated story summaries using OpenAI
- Full story expansion and narration
- Configurable voice, length, and background music
- Credit-based model (free and paid options eventually) 
- Community sharing and private library
- Secure user authentication and data storage (Firebase)

# Tech Stack
- **Frontend:** React / React Native Web
- **Backend:** Firebase Functions, OpenAI API, Google TTS
- **Database:** Firestore
- **Storage:** Firebase Storage
- **Payments** Stripe (via Firebase)
